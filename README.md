# Calculadora Web

Calculadora funcional feita com HTML, CSS e JavaScript puro — sem frameworks.

## Funcionalidades
- As 4 operações básicas (somar, subtrair, multiplicar, dividir)
- Histórico da operação no ecrã
- Botão de apagar (AC) e retrocesso
- Proteção contra divisão por zero
- Suporte ao teclado (digits, +, -, *, /, Enter, Backspace, Escape)

## Como usar
Abre o ficheiro `index.html` diretamente no browser — não precisa de servidor.

## Tecnologias
- HTML5
- CSS3
- JavaScript (Vanilla)
